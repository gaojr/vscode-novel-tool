## Features

- 全角转半角功能
  - 以下全角字符会被转换为半角字符:
  ```
  　＂＃＄％＆＇（）＊＋，－．／０１２３４５６７８９＜＝＞＠ＡＢＣＤＥＦＧＨＩＪＫＬＭＮＯＰＱＲＳＴＵＶＷＸＹＺ［＼］＾＿｀ａｂｃｄｅｆｇｈｉｊｋｌｍｎｏｐｑｒｓｔｕｖｗｘｙｚ｛｜｝～
  ```
- 删除行末空白
- 删除行首空白
- 增加空行
